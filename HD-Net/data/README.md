# Data

This repository does not include source datasets or private subject data.

Place prepared NPZ files here by default:

```text
data/hdnet/train.npz
data/hdnet/val.npz
data/hdnet/test_calbased.npz
data/hdnet/test_calfree.npz
```

Expected keys include `Signals`, `SBP`, `DBP`, `Age`, `Height`, `Weight`, and `Gender`.
